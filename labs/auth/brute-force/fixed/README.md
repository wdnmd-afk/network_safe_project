# 暴力破解修复版

修复版模拟登录失败保护：达到连续失败阈值后触发节流，并停止继续检查后续候选口令。

## 行为

- 只检查虚拟教学账号 `training-user`。
- 接收最多 5 个候选口令。
- 连续失败阈值为 3。
- 达到阈值后返回 `brute-force-rate-limit-blocked`。
- 正常单次正确候选口令仍返回 `brute-force-normal-login-accepted`。

## 修复重点

修复版强调失败计数和节流的边界：系统不能在已经达到失败阈值后继续检查后续候选口令，否则候选列表仍可能命中弱口令。

生产系统还需要结合验证码、MFA、IP / 设备风险评分、账户锁定策略、告警和审计。
