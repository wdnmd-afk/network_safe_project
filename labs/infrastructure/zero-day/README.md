# 零日漏洞利用

## 场景目标

通过固定事件响应案例学习未知漏洞下的暴露面收敛和补偿控制。

本实验只使用固定案例 `synthetic-unknown-exploit-signal`，用于观察关键资产缺少分层防御、行为检测和快速隔离能力，并对比漏洞版与修复版的后端判定、学习信号和统一事件日志。

## 前置条件

- 本机前端与后端服务已启动。
- 使用本项目本机演示账号登录。
- 不需要外部目标、第三方服务、真实凭据或真实业务数据。

## 使用方式

1. 访问 `/labs/infrastructure/zero-day/vuln`。
2. 使用固定案例和默认控制策略观察风险被接受。
3. 切换到 `/labs/infrastructure/zero-day/fixed`，观察高风险动作被阻断。
4. 选择“暴露面已收敛并监控”，确认正常受控流程仍可继续。
5. 在实验详情或账户中心复盘统一事件日志。
