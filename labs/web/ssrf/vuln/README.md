# 漏洞版说明

漏洞版模拟一个运营素材 URL 预览接口：

1. 用户提交一个 URL。
2. 后端只检查 URL 格式。
3. 后端没有限制目标主机是否为公开白名单。
4. 当目标是内部元数据地址时，仍然返回虚拟资源表中的内部模拟响应。

当攻击样例 `http://169.254.169.254/latest/meta-data/iam/security-credentials/demo` 被提交时，漏洞版会返回内部模拟元数据。

关键观察点：

- 目标主机是链路本地地址。
- 目标不属于公开白名单。
- 统一事件日志会记录 `ssrf-internal-metadata-exposed`。

本实现不发起真实 HTTP 请求，只用于观察“服务端代请求目标未受限”的风险链路。
