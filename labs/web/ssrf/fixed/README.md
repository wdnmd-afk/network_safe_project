# 修复版说明

修复版在模拟抓取资源前执行以下校验：

- 只允许 `http:` 和 `https:` 协议。
- 只允许公开白名单主机 `safe-mart-cdn.local`。
- 阻断回环地址、私有地址、链路本地地址和内部域名。
- 不执行真实网络请求，只查询内置虚拟资源表。

当同一个受控攻击样例提交到修复版时，接口返回 `ssrf-private-target-blocked`。

真实生产中还需要继续补充：

- DNS 解析后再次校验 IP。
- 禁止或严格校验重定向链路。
- 出站代理和网络层 egress 策略。
- 请求超时、响应大小和内容类型限制。
- 访问审计和异常目标告警。
