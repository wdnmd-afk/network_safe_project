# 信息泄露修复版

## 修复策略

修复版诊断接口只允许普通用户读取公开报告，例如：

```text
public-status
```

当用户提交调试报告样例：

```text
debug-diagnostics
```

后端会返回 403，并给出通用阻断说明，不返回堆栈、配置、token 形态或内部路径。

## 观察重点

- 后端决策为 `blocked`。
- 学习信号为 `info-disclosure-debug-data-blocked`。
- 页面不展示调试报告字段。
- 统一事件日志记录防御阶段、报告 key 摘要和阻断信号。
