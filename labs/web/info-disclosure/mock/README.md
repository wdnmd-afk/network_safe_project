# 信息泄露虚拟数据

本实验使用后端内置虚拟诊断报告表，不读取真实系统信息。

## 公开报告

```text
public-status
```

用于模拟普通用户可见的服务状态摘要。

## 调试报告

```text
debug-diagnostics
```

用于模拟错误配置下普通用户可读取的内部调试报告。内容全部为教学占位：

- 堆栈摘要是模拟错误文本。
- 配置键名是演示字符串。
- token 形态是占位格式，不是真实 token。
- 内部路径是教学路径，不是本机真实路径。

## 日志约束

事件日志只保存报告 key、是否敏感报告、字段数量和学习信号，不保存完整调试报告内容。
