# NoSQL 注入漏洞版

## 目标

漏洞版展示用户输入影响虚拟文档查询语义后的风险。

## 行为

- 使用内存虚拟优惠券文档集合。
- 正常关键词只返回公开优惠券。
- 固定受控风险样例触发查询范围扩大信号。
- 返回的隐藏记录必须是教学虚拟记录。
- 事件日志只记录风险类别、输入长度、结果范围和学习信号。

## 禁止行为

- 不连接真实 NoSQL 服务。
- 不执行任意对象查询。
- 不实现通用操作符执行器。
- 不返回真实敏感数据。
- 不保存完整查询结构。

## 入口

页面：

```text
/labs/web/nosql-injection/vuln
```

API：

```text
POST /api/labs/web/nosql-injection/vuln/search
```
