# 针对性误判观察版

## 目标

本目录用于后续记录鱼叉式钓鱼的误判观察版说明。当前只建立文档占位，不提供页面、API、脚本或真实投递能力。

后续漏洞版只允许通过固定虚构线索卡观察：

- 针对性上下文如何提高可信感。
- 角色权威和紧急压力如何压缩核验流程。
- 付款、权限、人事或供应商变更请求如何绕过正式审批链。
- 缺少可信通道二次确认时可能出现的错误放行。

## 当前状态

- 当前没有前端入口。
- 当前没有后端接口。
- 当前没有事件日志写入。
- 当前没有自动化脚本。
- 当前不提供完整邮件正文、IM 对话、标题库、附件诱导文案或真实链接。

## 安全边界

- 不采集真实目标画像。
- 不使用真实姓名、邮箱、手机号、公司、部门、职位或社交账号。
- 不发送真实邮件、短信、消息或链接。
- 不收集真实凭据、验证码、Cookie、token、付款信息或附件。
- 不提供 `exploit.py`、投递脚本、模板生成脚本或攻击脚本。
