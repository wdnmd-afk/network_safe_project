# 流程核验复盘版

## 目标

本目录用于记录鱼叉式钓鱼的流程核验复盘版说明。当前已提供前端固定案例工作台、后端受控 `review` API 和本机只读一致性验证，不提供攻击脚本。

当前修复版重点展示流程防御，而不是只做单个线索判断：

- 角色核验。
- 可信通道二次确认。
- 财务、人事、供应商和工程权限审批链。
- 供应商主数据变更流程。
- 临时访问最小授权和有效期。
- 隔离、举报和事件复盘。

## 当前状态

- 当前已有前端入口：`/labs/social/spear-phishing/fixed`。
- 当前已有后端受控接口：`POST /api/labs/social/spear-phishing/fixed/review`。
- 当前接口已写入统一事件日志安全摘要。
- 当前只有本机只读一致性验证脚本，不提供攻击自动化脚本。
- 当前不处理真实人员、真实组织、真实邮箱、真实链接或真实附件。

## 安全边界

- 不连接第三方邮件、短信、社交平台、CRM、HR、IM、企业通讯录、投递服务或收件箱服务。
- 不保存真实业务材料、真实凭据、验证码、Cookie、token 或付款信息。
- 当前页面和 API 只能读取固定 `caseKey` 和 `verificationPolicyKey`。
- 当前事件日志只能记录固定 key、风险类别、决策、建议动作和学习信号。
