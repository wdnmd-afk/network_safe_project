# 流程核验复盘版

## 目标

本目录用于后续记录鱼叉式钓鱼的流程核验复盘版说明。当前只建立文档占位，不提供页面、API、脚本或自动化验证。

后续修复版应重点展示流程防御，而不是只做单个线索判断：

- 角色核验。
- 可信通道二次确认。
- 财务、人事、供应商和工程权限审批链。
- 供应商主数据变更流程。
- 临时访问最小授权和有效期。
- 隔离、举报和事件复盘。

## 当前状态

- 当前没有前端入口。
- 当前没有后端接口。
- 当前没有事件日志写入。
- 当前没有自动化脚本。
- 当前不处理真实人员、真实组织、真实邮箱、真实链接或真实附件。

## 安全边界

- 不连接第三方邮件、短信、社交平台、CRM、HR、IM、企业通讯录、投递服务或收件箱服务。
- 不保存真实业务材料、真实凭据、验证码、Cookie、token 或付款信息。
- 后续若提供 API，只能读取固定案例 key 和固定检查策略 key。
- 后续事件日志只能记录固定 key、风险类别、决策和学习信号。
