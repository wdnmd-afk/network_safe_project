# 固定虚构案例说明

## 目标

本目录用于后续放置鱼叉式钓鱼固定虚构案例说明。当前只建立文档占位，不放置真实邮件、真实聊天记录、真实联系人信息、真实组织结构或可投递模板。

## 允许的素材类型

- 固定案例 key。
- 虚构角色标签。
- 风险标签。
- 线索卡摘要。
- 流程节点说明。
- 防御动作清单。
- 复盘问题。

## 禁止的素材类型

- 禁止放置真实姓名、邮箱、手机号、公司、部门、职位、社交账号或组织关系。
- 禁止放置完整邮件正文、完整 IM 对话、可复制标题、可投递附件名或真实链接。
- 禁止放置可投递标题库、正文模板库、IM 话术库、附件诱导文案、跟踪链接或群发脚本。
- 禁止放置第三方平台导出的通讯录、CRM、HR、IM 或收件箱数据。

## 当前固定案例方向

- `executive-invoice-approval`
- `vendor-payment-change`
- `engineering-access-request`
- `hr-benefit-personalized`

这些方向已在 `labs/social/spear-phishing/docs/fixed-cases.md` 中补充为固定虚构案例卡。它们只表示后续学习案例类别，不是当前已实现页面字段，也不是可投递内容。
