# 高权威误判观察版

## 目标

本目录用于记录捕鲸攻击漏洞版的 planned 说明。后续只能通过固定虚构线索卡观察高权威、保密理由、重大业务事件和时间压力如何诱发越级审批、可信通道缺失或例外流程失守。

## 当前状态

当前未实现页面、API、数据库写入、事件日志写入或脚本。本目录仅作为 `social.whaling` planned 元数据中的标准目录占位。

## 允许观察的内容

- 高层授权压力如何让接收者过度相信请求。
- 董事会、法务、外部顾问或并购语境如何制造保密压力。
- 大额付款、临时资料室访问和例外审批如何压缩核验时间。
- 缺少可信通道回拨、双人复核或例外冻结时会出现哪些学习信号。

## 禁止内容

- 不写完整邮件正文、IM 对话、会议邀请、可复制标题或付款指令。
- 不采集真实高管、组织结构、汇报关系、公开资料、邮箱、手机号或职位链路。
- 不生成投递模板、附件诱导文案、跟踪链接、群发脚本或攻击脚本。
- 不提供 `exploit.py` 或任何可面向第三方目标使用的能力。
