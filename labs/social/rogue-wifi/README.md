# 钓鱼 WiFi

## 场景目标

通过固定无线网络卡片学习同名热点、门户诱导和可信网络核验。

本实验只使用固定案例 `synthetic-evil-twin-network`，用于观察用户和设备仅依赖 SSID 名称判断网络身份，并对比漏洞版与修复版的后端判定、学习信号和统一事件日志。

## 前置条件

- 本机前端与后端服务已启动。
- 使用本项目本机演示账号登录。
- 不需要外部目标、第三方服务、真实凭据或真实业务数据。
