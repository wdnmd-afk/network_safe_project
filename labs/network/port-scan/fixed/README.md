# 修复版观察

## 1. 定位

修复版用于展示最小暴露面治理后的虚拟资产状态。

本目录当前只放置说明文档，不包含可运行代码。

## 2. 计划观察点

当前修复版受控 API 使用固定虚拟目标展示：

- 只保留必要业务端口。
- 管理面改为受限或内网专用。
- 多余服务被下线。
- 资产清单与暴露面审计形成闭环。

## 3. 修复思路

修复版重点展示：

- 最小暴露面。
- 网络访问控制。
- 管理端口隔离。
- 服务指纹收敛。
- 资产清单定期审计。
