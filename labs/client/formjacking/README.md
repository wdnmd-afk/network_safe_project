# Formjacking

## 场景目标

通过固定结账页提交摘要观察脚本篡改与表单目标校验。

本实验只使用固定案例 `synthetic-checkout-target-change`，用于观察敏感页面信任未受约束的第三方脚本和提交目标，并对比漏洞版与修复版的后端判定、学习信号和统一事件日志。
