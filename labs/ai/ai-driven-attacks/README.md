# AI 驱动攻击

## 场景目标

本实验只使用固定案例 `synthetic-automated-attack-chain`，用于观察自动化系统拥有过宽工具权限且缺少人工确认和审计，并对比漏洞版与修复版的后端判定、学习信号和统一事件日志。
