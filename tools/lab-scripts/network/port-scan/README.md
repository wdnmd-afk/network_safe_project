# 端口扫描暴露面脚本目录

## 1. 当前状态

当前目录用于记录脚本边界，并提供一个本机只读一致性验证脚本。

本阶段不创建：

- `exploit.py`
- 真实端口扫描脚本

当前可执行脚本：

- `verify.ts`：只读取本仓库内端口扫描元数据、文档和实现文件，确认入口、自动化证据和安全边界一致。

当前不提供 exploit.py 或真实扫描脚本。

## 2. 后续脚本边界

如果后续需要脚本，只允许：

- 读取本仓库内固定元数据和文档。
- 输出本机受控学习信号。
- 在明确执行文档约束下调用本项目 localhost API。

禁止：

- 接收任意主机、网段或端口范围。
- 扫描局域网、网段或外部 IP。
- 调用系统网络探测命令。
- 使用原始 socket 扫描。
- 并发探测或压力探测。
- 输出可复用到外部目标的扫描结果。
