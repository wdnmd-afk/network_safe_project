## 2. 后续脚本边界

如果后续需要脚本，只允许：

- 读取本仓库内固定元数据和文档。

禁止：

- 接收任意域名、DNS 服务器、IP、代理、网络接口或端口参数。
- 请求真实外部 DNS、DoH、DoT 或公共解析服务。

- 实现真实 DNS 投毒、DNS 劫持、DNS 隧道、流量转发或中间人代理。
- 输出可复用到外部目标的真实解析或劫持结果。
